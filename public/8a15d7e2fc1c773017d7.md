## はじめに

Microsoft が [Intelligent Terminal version 0.1 を発表しました](https://devblogs.microsoft.com/commandline/announcing-intelligent-terminal-version-0-1/?WT.mc_id=DT-MVP-5004827)。
これは Windows Terminal をベースにした、エージェント機能を組み込んだ実験版のターミナルです。

普通のターミナルは「コマンドを打つ場所」ですが、Intelligent Terminal はそこにエージェントの操作面が加わります。
エラーの説明、修正の提案、過去セッションの再開まで、ターミナルの中でそのまま扱えるのが特徴です。

この記事では、まず Intelligent Terminal の基本を整理し、そのあと実際にインストールして試した内容を書きます。

## Intelligent Terminal とは

Intelligent Terminal は、Windows Terminal の実験的な派生版です。
単に見た目が変わるだけではなく、**エージェントとターミナルを同じ画面でつなぐ**のが大きなポイントです。

主な機能は次のとおりです。

| 項目 | 内容 |
|------|------|
| 🧭 エージェントペイン | シェルの右側や下側に、エージェント用のペインを開けます |
| ⚠️ エラー検出 | コマンド失敗を検知して、修正のきっかけを出せます |
| 🗂️ セッション管理 | 過去のエージェントセッションを一覧して、途中から再開できます |
| ❓ コマンドパレット | 自然言語の指示を入口にして、エージェントを起動できます |
| 🤖 エージェント切り替え | GitHub Copilot CLI だけでなく、ACP 対応エージェントも使えます |

デフォルトのエージェント体験は GitHub Copilot CLI です。
つまり、最初から「Copilot と一緒に使うターミナル」として始められます。

ショートカットも分かりやすくて、エージェントペインの表示・非表示は `Ctrl+Shift+.`、フォーカス切り替えは `Ctrl+Shift+I` です。
セッション管理パネルは `Ctrl+Shift+/` で開けます。

## インストールしてみた

今回は Microsoft Store から入れて試しました。
WinGet でもインストールできます。

```powershell
winget install Microsoft.IntelligentTerminal
```

インストール後に起動すると、下部にエージェントペインがあり、通常の PowerShell 画面と並んで動きます。
最初は「ターミナルの中に Copilot がいる」という見た目で、かなり分かりやすいです。

![Intelligent Terminal のエージェントペイン](https://raw.githubusercontent.com/tomokusaba/qiita/main/public/images/8a15d7e2fc1c773017d7/count-tmp-files.png)

## 自然言語でコマンドを作ってくれる

試してみて一番分かりやすかったのは、自然言語でそのまま指示できる点でした。

たとえば、

> tmpファイルはいくつある？

と入力すると、エージェントが意図をくみ取って PowerShell コマンドを提案してくれます。

実際には、こんなコマンドが生成されました。

```powershell
(Get-ChildItem -Path . -Filter *.tmp -File -Recurse -ErrorAction SilentlyContinue | Measure-Object).Count
```

これで、カレントディレクトリ以下の `.tmp` ファイル数を数えられます。
`Get-ChildItem` と `Measure-Object` を自分で組み合わせなくても、自然文からそのまま形にしてくれるのが便利です。

![自然言語からコマンドを作る様子](https://raw.githubusercontent.com/tomokusaba/qiita/main/public/images/8a15d7e2fc1c773017d7/count-tmp-files.png)

## 過去のセッションを選び直せる

もう 1 つ便利だったのが、セッション管理です。
GitHub Copilot CLI がベースなので、過去にやり取りしたセッションが一覧で出てきます。

これにより、いったん終わった作業を後から掘り返して続けることができます。
「さっきの調査の続きを見たい」「別のタブで走らせたタスクに戻りたい」といった場面でかなり助かります。

実際にセッションを開くと、過去の作業内容がそのまま残っていて、どのタブで何をしていたかも追いやすいです。
Copilot のバージョン表示や AI Credits の残量も見えるので、今どのエージェント体験にいるのかが分かりやすいのも良いところでした。

![開いたエージェントセッション](https://raw.githubusercontent.com/tomokusaba/qiita/main/public/images/8a15d7e2fc1c773017d7/opened-session.png)

![過去のセッションを選べる画面](https://raw.githubusercontent.com/tomokusaba/qiita/main/public/images/8a15d7e2fc1c773017d7/session-picker.png)

## まとめ

Intelligent Terminal は、Windows Terminal にエージェント操作を自然に重ねた実験版でした。

普通のターミナルの延長で使えるのに、
自然言語でコマンドを作れたり、過去のセッションを呼び出せたりするので、かなり「次のターミナル」という印象があります。

まずは実験版として触ってみる価値は十分ありました。
